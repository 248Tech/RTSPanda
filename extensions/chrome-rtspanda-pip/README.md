# RTSPanda Camera PiP Chrome Extension

This extension toggles Picture-in-Picture for the currently viewed RTSPanda camera page.

It is PiP-only and does not open webpage popups.

## Install

1. Open Chrome and go to `chrome://extensions`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select this folder (`chrome-rtspanda-pip`).

## Use

1. Open a RTSPanda camera page (`/cameras/{id}`).
2. Click the extension icon.
3. Camera feed opens in PiP.

PiP windows are movable/resizable and remain visible while browsing other tabs.

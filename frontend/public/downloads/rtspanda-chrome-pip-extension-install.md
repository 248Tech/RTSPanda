# RTSPanda Chrome PiP Extension Install

1. Download `rtspanda-chrome-pip-extension.zip`.
2. Unzip the archive.
3. Open `chrome://extensions`.
4. Turn on `Developer mode`.
5. Click `Load unpacked`.
6. Select the extracted extension folder.

Use:

1. Open a RTSPanda camera page (`/cameras/{id}`).
2. Click the extension icon to toggle Picture-in-Picture.
